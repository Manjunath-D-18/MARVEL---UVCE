Projects

IoT-Based Flood Monitoring and Alert System Using ESP32

I built a real-time flood monitoring system using an ESP32 integrated with an ultrasonic sensor to accurately measure rising water levels. The system features a multi-tier alert mechanism that triggers visual indicators through LEDs, audio warnings using a buzzer, and displays live water-level data on an LCD screen. For high-risk conditions, it automatically activates a servo motor to simulate floodgate control and sends real-time SMS alerts as part of its IoT-based functionality. The entire setup was designed to be cost-effective, scalable, and practical for deployment in both urban and rural flood-prone areas. After thorough hardware testing, the system consistently demonstrated accurate water-level detection and reliable alert responses across all defined threshold states.

LI-FI Based Data Transmission (Ongoing)

My team and I developed a Li-Fi prototype using ESP32, Arduino Uno, an LED for light-based data transmission, and an LDR sensor to detect variations in light intensity. In the first stage, we implemented Morse code communication, where fluctuations in light were decoded into readable text and displayed on an LCD screen. Building on that, we moved to a second stage focused on creating an encrypted bi-directional communication link between two ESP32 modules, integrating AES encryption to ensure secure data transfer. The overall system was designed as a secure, low-interference communication model to demonstrate Li-Fi’s advantages over traditional RF-based wireless systems.
